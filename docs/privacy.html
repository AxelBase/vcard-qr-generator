Privacy Policy

Privacy Policy

Effective Date: November 20, 2025

We Collect Absolutely Nothing

The vCard QR Generator is a completely client-side web tool. Every single operation — from filling the form to generating the vCard string and rendering the QR code — happens exclusively in your browser.

We do not collect, store, transmit, log, or process any personal information whatsoever. This includes:

  • Names, phone numbers, emails, addresses, or any contact details you enter
  • IP addresses, browser fingerprints, or geolocation data
  • Usage statistics, analytics events, or crash reports
  • Cookies, localStorage persistence (beyond the current session), or tracking pixels

How the Tool Actually Works

All processing is performed locally using pure JavaScript:

  • vcard-creator builds the vCard 3.0 strings in memory
  • Qrious draws the QR code directly on an HTML <canvas>
  • Download buttons trigger saveAs() via file-saver — no server round-trip

Your data never leaves your device — not even temporarily.

No Third-Party Services

We do not use:

  • Google Analytics, Plausible, Fathom, or any analytics provider
  • Advertising networks, affiliate pixels, or social media widgets
  • External CDNs that track users (all libraries are bundled locally)

Security Is Inherent

Because there is no backend, there is no database to breach, no logs to leak, and no API keys to steal. The only code that runs is what you see in your browser’s developer tools.

Children’s Privacy

This tool is safe for all ages. No personal information is ever requested or stored.

Compliance by Design

By collecting zero data, we are inherently compliant with GDPR, CCPA, CCPA/CPRA, LGPD, and all other privacy regulations.

Changes to This Policy

If we ever change this policy (extremely unlikely), the new version will be posted here with an updated “Effective Date”.

Your privacy is not a feature — it’s the entire architecture.
Thank you for trusting a truly private tool.
